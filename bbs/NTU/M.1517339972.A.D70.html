原文在陳品私人臉書( https://goo.gl/fiV3P2 ) 不過本人開地球也分享到台大學生交流板了 我想說這篇文章還挺不錯的，就分享過來了XD -- 當學代遇到的人 ─ 臺大教務長郭鴻基 　 寫這篇主要是想紀錄一下遇到的事情，剛好任內比較多和教務處的互動，雖然不能代表全部 ，但應該某種程度上反映了一些人在所謂「學生自治 = 跑各處室」的領域裡會遇到的各種 好的壞的神奇的事情。 =============== 　 說到台大教務長郭鴻基，第一個印象就是喜歡講話。對，沒錯，他非常愛講話，學生去找他 討論事情的時候，幾乎都得先聽他講話，而且不管你本來想問他的問題是什麼，他都能從自 己的教學理念開始談起，然後講學生面對變化多端的社會應該培養哪些能力，再講他對台大 課程改革的遠大擘劃，整個過程大概會經歷十幾分鐘，最後再稍微帶回你本來要談的問題， 用一分鐘做個總結(如果有回應到的話啦)。 在教務會議的時候就更生猛了，他在主席致詞的內容可謂上天下地，還有從各種經典裡引來 的高級文言文，大大強化了他論述教育理念的說服力，幾乎都要講個三四十分鐘，把在場師 生都好好勉勵一番，才心滿意足的進到真正的議程討論，喜歡講話的程度可見一斑。 至於教務長這個所謂的教育理念到底是什麼呢？雖然不敢自稱全盤了解，但畢竟也聽了有點 非常多次，我猜大概可以歸納成一個核心理念，就是「有用」，也就是要讓學生都去學用得 到的東西。具體來說，他把本來上下學期各三學分的國文課改成只要修一個就好，還放寬了 通識的修課限制，這也讓前年的學生會福利部刷了不少 credit。 他說我們應該要減少學生浪費在沒用廢課的時間，取而代之的是更實用的課，比方說大一國 文改開寫作班，通識可以修入門電腦程式課，然後弄了幾個很像超越文理組的跨領域學程， 還有高級的歐洲語學程，讓學生畢業後都能做個有用的人。 =============== 　 當然，說是一回事，做又是另一回事，只聽一個人講的道理是不夠的，我們還要檢視他在面 對事情的時候如何反應，看看其言其行是不是有符合才行，所以就來講幾個小故事吧。 　 一、 最早和教務處事務搭上的原因，其實是在學期初的時候，導因於再前一個學期的課：數位應 用程式設計，一門在課程版上討論熱度超高的新開課程，傳說中老師幾次上課遲到，期末作 業的 spec 會忽然大改，還會臨時冒出本來不在期初宣布的課堂規劃內的成果發表會，因故 不能出席還會被扣分，卻只是為了能向長官誇耀課辦得很好。種種怪現象讓我當時決定公開 反推開課老師，順便鼓勵大家退選那個很愛把課名取得很炫砲(eg:嵌入式微型電腦互動設計 )的 CS+X 系列課程，結果就引出當事老師出來滅火兼想找我私下約談。 不談不知道，一談才發現，唉呀，這個老師也是有那麼一點點情有可原，其實整個系列課程 都是教務長遠大擘劃下的一部分，也就是文組學寫程式變有競爭力的那部分，但他一方面又 好大喜功，才要老師臨時硬塞個期末成發來收割一波；另一方面又吝嗇給資源，想要開大班 課又不配助教名額，才會讓作業啥的規劃都爆炸成一團亂，也間接釀成這第一次開課就評價 全毀的大悲劇。不過當我因為這件事第一次主動去和教務長聊天的時候，他本人好像倒是挺 滿意的，還說改革就是要先實驗，等到有成果的時候資源自然會給到位，只不過這些拿去炫 耀的成果都是靠壓榨修課學生來的就是了。 當時負責這個計畫，或者說也稱得上老闆的還有副教務長康仕仲，也是對教務長的這個先求 表現後給資源的規劃深表認同，還把自己的一門叫資料視覺化的課放進去變合授。 結果這學期就停開了。 　 二、 教務長自認很滿意的另一件事情，就是他讓本來的大一英文變成了「大一外文」，學生可以 修各種外語，不限定英文，再搭配他發明的歐洲語學程獲得畢業證書上的認證，這樣學生就 會變得更有競爭力。但我聽他這樣講的時候怎麼都覺得奇怪，我就記得明明只能修英文阿， 哪來的別種外語？去問了才發現這個條款有個但書，各系可以自行規定採計哪些語言，然後 大部分的系都不約而同的選了...英文，結果就是好像沒什麼改變的樣子。 大一英文不能換就算了，反正橫豎都是上課，不過當時正好碰上政大正在廢畢業英文檢定( 已經正式廢除了)，而這方面台大一直用很垃圾的方法唬爛學生，校方會說我們沒有英檢門 檻喔，我們只是有兩門零學分的必修叫進階英文，然後這個課剛好可以用英檢來抵免，不過 因為要程度分班適性教育，所以又規定修課之前一定要考英檢才能用成績分班，但如果你幸 運考過那就恭喜不用修了，總之就是拐彎抹角的給你一個跟畢業英檢門檻有八成像的零學分 必修啦。 對於這個變相的逼學生考試的假必修，其實一直以來都有學生向校方表達應該廢除的意見。 那反正我去的那次呢，教務長就說，他其實也是很樂見大家討論啦，畢竟以後是國際化的社 會，大家應該學習對自己領域有用的語言，我就問他那是不是可以在教務會議上處理，就把 這個零學分爛必修改成選修了吧，他也沒表示反對。於是我就跟當時的學生會長呂姿燕講感 覺可以試試看，她就著手寫修法規的提案單，然後我們就四處拜託有席次的學生代表出席連 署。 這實在是個很燒人品的過程，要求另一個大學生早上九點爬起來聽教務長講大道理，實在是 不太讓人感興趣的事情，但總之當時來了學生會長、議長、研協會長、各院學生會長加一加 大概有機會搆到提案門檻，現場好像是差三個人簽名，但大部分出席的老師主任都滿排斥幫 忙連署的，大概是怕惹麻煩(?)吧，幸好最後有人超好的應力所、城鄉所、台文所的老師幫 忙簽字，成功卡上了臨時動議的提案。 然後就被封殺了。 教務長接過祕書遞上去的提案單，發現連署人數居然夠，眉頭皺了一下，說：「很抱歉耽誤 各位老師的時間，這裡有學生的臨時動議，要修改這個進階英文實施辦法。」然後沉默了一 陣子接著說：「我想這個還是要照流程來，學生的意見我們聽到了，這個法案就先送到共教 委員會(某個學生沒有席次的委員會)去討論，下次再排案吧。」接著不管我們如何質疑這樣 恐怕沒辦法採納學生意見，教務長還是堅持不可以現場討論，強勢宣布散會。 之後才知道，原來他本來是不認為我們能湊足連署人數，才會表現得樂觀其成的樣子，也坦 承對於這件事內心根本堅決反對，理由是他很想把大一英文成功推成大一外文，所以如果留 著畢業英文門檻的話就可以減低來自其他教授的阻力。標準「朕不給的，你不能要」的心態 ，鴻基你不給就早說嘛，學生們也不必辛苦折騰早起來演這場戲了，而且瑞凡，大一英文實 際上也沒有真的變成超有競爭力的大一外文啊！ 　 三、 還有另一件讓我介懷超久的事情。背景大概是這樣的，很久很久以前教務處曾經開放資料庫 讓大家查某一門課老師的給分情形，也就是每個等第的人數，然後就有熱心的學生把它包成 一個超讚的服務，選課的時候可以直接查歷年的 GPA 變化圖，旁邊還對照課程網頁和 PTT 的評價文，省了很多來回翻查的時間。但這個服務在 104 年後面就沒有更新資料，原因是 教務處把撈資料的 API 給鎖起來了。 對學生這麼方便的東西不見了當然是要去爭取一下，我就在教務座談會前先寫信去詢問能不 能重開，當天教務長跑來說，其實呢他也沒想鎖查成績分布的 API ，只是有某些勢力暗中 反對還大力施壓，教務處才忍痛封起來的，還叫資訊組主任來和我說明苦衷(?)，說這都是 學校計中網路組組長林宗男一個人的意思，說這個資料放出來有個資疑慮，所以他身為校級 資訊網路主管機關發函來一定要禁止繼續開放資料blabla。 到這邊聽起來完全是個鬼扯的故事了，我想說好吧，既然如此就再跑一趟計中聽看看一門課 的成績統計究竟能有三小個資疑慮。那反正和林宗男組長見面當天我也直接說明來意，我覺 得課的成績分布根本不能對應到學生的識別資訊，講個資疑慮顯然是在鬼扯，應該要重新開 放 API 存取。 一開始林組長當然就有稍微辯解來辯解去的，說什麼人數少的課會能識別個人啦，還說什麼 就算成績分布不會洩漏學生的個資但那也是老師(蛤?)的個資啦，但畢竟是站不住腳的辯詞 ，所以後來只好開大絕說：「好啦，我承認成績不是個資，但就是有很多老師會不爽阿，如 果他們去盧小教務處的職員你能負責嗎？」，然後說他頂多同意可以有選項讓老師主動開啟 ，我想想覺得也是有點道理，就問說那能不能預設是開啟，有選項讓小心眼的老師可以關掉 他們就不會去煩教務處了，林宗男組長說，他覺得這樣可以，看教務處。 到這邊聽起來很像是有共識了，回去和教務處資訊組的人講這個好消息的時候他們也說OK， 只要上面來一紙公文就可以開放啦，但就在一切要皆大歡喜圓滿團員的時候，起初表現得很 無關的郭教務長鴻基說話了，他說了一句很厲害的話：「很感謝同學們關心學校的事情，但 你也知道的，我們的學生真的很需要一些有幫助的課程，我也很努力替同學規劃，在這個改 革的『節骨眼』，我不能讓別的事情節外生枝，引起老師們的不滿反彈，所以這個還是緩一 緩，現在先不要啦。」 總之呢，這就是另一個跑斷腿的傻子跟一開始看起來樂觀其成但其實根本堅決反對可是讓你 去繞一繞消耗精力也不錯的聰明人的相忍為改革的故事，至於那個成績 API 後來究竟如何 呢？大概因為開放這個沒辦法讓學生變得更有用吧，所以儘管如今又一個學期過去了，計中 網路組組長都換了，教務長眼看也該換了，感覺卻還是遙遙無期。 =============== 　 以上大概就是一些跟教務處有關的小故事們，其實教務處的職掌還滿寬的，包括這學期的那 些操場被輾爛的故事啊，其實決定出借的體育組的上司也是教務處；然後爛了很多年未解的 服務學習也是教務處負責規劃但一直推諉給負責執行的學務處。凡此種種都和學生生活滿有 關係的，但學生仍然面臨著被幼體化，被當作打發一年就好反正會換人的過客，被放在行政 體系裡面繞圈圈的無力處境。不能說一直以來，免得又被說代言了誰，但至少在一個學期中 ，以和教務處的互動來說，某種程度上，確實是能做成什麼改變還比較稀奇呢。
