標題受夠關於年輕人的新聞或假新聞了嗎?
時間Tue Mar 7 15:29:58 2017
那篇什麼薪酸的新聞就不管了。 現在太多莫名奇妙的新聞，但很少有文章好好討論年輕人要怎做。看到很多學弟妹徬徨，我覺得NTU板上有多ㄧ點實實在在的建議會比較好，所以來獻醜個。 第ㄧ點，建議學弟妹先從想成為社會上哪個角色來思考，尤其先用社會上既有的結構出發，因為改變框架比較難，後面會討論。譬如，理工的就要想想自己要成為工程師，管理者，業務工程師，還是研究者。並讓自己的學習開始像金字塔般有聚焦，你會比較容易成為專業人士。 第二點，要多用Linkedin, 學校演講，同學老師或你所有的連結去了解要怎麼成為那些角色或所需要的專業技能要怎麼培養。ㄧ方面，社會上騙人的新聞和資訊太多了，很多都是付錢為了詐取不懂的年輕人所產生的新聞。 如果團結，有ㄧ個資訊和人脈網，會比較容易成功。 另ㄧ方面，藉有了解別人的職涯規劃或經驗，你可以多想想自己該怎麼做或想不想要。尤其，每個階層的都要看，因為平常我們在學校遇到的都是20幾歲，要不然就是教授，其實久而久之，思考和看的都會比較窄。 第三點，要盡量多跟國際接軌，不然容易被台灣的眉角和公司綁死。這點雖然比較難，但我們還是要盡量嘗試。資工的也許可以從開源的專案或討論區開始，多增加自己的能見度，實力，和英文。其他學院的再請其他系學長姐補充了。對於小國而言，唯有多交流，在國內，你才有能力和眼光和國外公司競爭，並讓台灣老闆不砍價。要出國的話，你也可以提早讓國外認同你，所以建議儘早做，有機會的話。 最後ㄧ點，如果你想做超越現有框架的事，你要開始找同伴，上級的認同者或投資者，並累積實力以改變框架。我常常聽到很多同學說，我想整合這幾個想法，取代現有的模式。這些都很好，也是我們年輕人該做的事。不過問題來了，要實踐，需要有人和你ㄧ起做，需要有人投資你，需要有上層幫你擋舊有的模式，這些都需要積累。如果你有很多想法，那最好趕快開始，不然等到你快出社會，又要工作，你很可能會來不及實現你的想法。所以盡早規劃吧! 以上只是我個人的建議，希望這個板可以有越來越多實質在幫助年輕人的建議，不然現在假新聞，假論文，假老師實在太多了。同學加油~ ※ 引述《Reewalker (報告饅頭班長不夠)》之銘言： : ※ [本文轉錄自 Reewalker 信箱] : 作者: isaluku (山本君) 看板: Gossiping : 標題: [新聞] 有夠「薪」酸！ 連台大研究所畢業生起薪都不到30K : 時間: Mon Mar 6 13:46:20 2017 : https://udn.com/news/story/9/2323744?from=udn-relatednews_ch1 : 有夠「薪」酸！ 連台大研究所畢業生起薪都不到30K : 國內青年低薪問題嚴重，連擁有「名校光環」的台大大學部，甚至研究所畢業生的首分工 : 作起薪，竟都不到30K。 : 根據yes123求職網的最新調查發現，如果是「台灣大學大學部」的畢業生，今年企業願意 : 提供的起薪，平均為2萬7908元，比起整體大學應屆畢業生平均的2萬6818元起薪，等於多 : 給4.1%的薪資（差距是1090元）。 : 如果是「台灣大學研究所」的畢業生，企業願意提供的起薪雖較高，平均達2萬9947元， : 比起平均的2萬8600元，等於多給4.7%的薪資（差距為1347元），但依舊連30K都不到。 : yes123求職網近日公布大學應屆畢業生對於首份工作「月薪期望值」的最新調查，平均值 : 落在3萬1284元，比起去年的3萬1538元，微幅減少254元，換算下來，降幅0.8%，但即使 : 大學應屆畢業生已經調降自我心中的月薪期望值，但企業給薪的水準仍是低於新鮮人的期 : 望。 : 看來，台灣青年低薪問題，短期內仍是無解。 : yes123求職網發言人楊宗斌憂心表示，從過去幾年來的期望值觀察，像前年是3萬1217元 : ，大前年落在3萬1180元，恐怕在畢業生心中，已形成3萬塊出頭的「起薪天花板」：初入 : 台灣職場，不敢奢望能拿到更高的月薪。 : 同時再以新鮮人的「學歷別」交叉分析，在「月薪期望值」部分：「學士」為2萬9863元 : ，「研究所」為3萬5672元。 : 只是從前述的月薪期望值來看，「學士」新鮮人為2萬9863元，與業者給的起薪2萬6818元 : 相比，仍然少了3045元，落差幅度有10.2%，等於打「九折」。 : 至於「研究所」新鮮人，月薪期望值為3萬5672元，與業者給的起薪2萬8600元相較，甚至 : 少掉7072元，「勞資雙方」的落差，更是高達了19.8%的幅度，等於打「八折」。 : 楊宗斌解讀這樣的現象，他表示，目前20到24歲的失業率為12.35%，大約每八個畢業生， : 就有一個待業中，但其實不少人新鮮人在應徵工作時，常常不滿意起薪，想要等待更多企 : 業的面試通知，最終當然影響媒合速度，拉長待業期，就某種程度上來講，算是間接導致 : 「高學歷、高失業率」的窘境。 : 上述「畢業生求職與校園徵才調查」，是由yes123求職網於2017年2月16日到3月1日，以 : 網路問卷進行抽樣，調查今年應屆畢業生，以及將(已)退伍、免役會員，同時具大專以上 : 學歷，有效問卷共1360份，信心水準為95%，誤差值為正負2.66%。 : 至於企業部分，有效問卷共928份，信心水準為95%，誤差值為正負3.22%。 ----- Sent from JPTT on my HTC_D626q. -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 101.14.115.21 ※ 文章網址: https://www.ptt.cc/bbs/NTU/M.1488871800.A.419.html
