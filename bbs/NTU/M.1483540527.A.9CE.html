※ 引述《wengeniusgod (wen12648)》之銘言： : ※ 引述《Kimg (Kimg)》之銘言： : : 這篇不是廢文 是認真想問 : : 很多課網路上都有同一個老師開的 : : 一模一樣課程內容的開放式課程 : : 這樣為什麼還要開課？ : : 學生在宿舍自己把影片念完 習題寫完 : : 不會的在問同學 : : 同學不會的也可以丟網路上一些社團找解答 : : 實在不懂為什麼還要開課 : : 把影片裡面一樣的東西在班到課堂上講 : : 我是覺得這樣是在浪費教授時間也在浪費學生時間 : 這是翻轉課程的概念吧 : 學生在家裡先把課程內容念完 : 在課堂上提出問題大家一起討論 : 或者一起做老師出的習題來練習 : 增加上課的效率 : 學生也可以自由選擇時間來學習課程 : 之後也可以看影片複習 : 但我自己是覺得這樣反而讓學生需要花更多時間在一門課上 : 課程影片少說都一個多小時 : 雖然看起來不多 : 但台大最低學分限制15學分 : 平時要做的事情真的不少 : 反而增加學生的壓力 : 此外如果學生偷懶沒看影片 : 反而上課的時候都不知道課堂在討論什麼 認真回一下這篇 XD，其實原 po 的問題確實很好， 有些課（不是全部）在傳統實體課上 90% 都是在聽教授講解， 以互動性來說跟看影片沒兩樣，而影片還可以挑時間播、重播等等， 只要授課老師認真關心影片品質，很容易就可以比實體課講授更有效率， 普及性也更高（假設老師願意的話）。而現在線上教育系統愈來愈發達， 作業繳交甚至小考什麼的都不是不可能，所以有些實體課存在的價值確實會降低。 這種情況下，開實體課唯一的理由就是「提供認證」吧， 但老實說這也不是理由，因為完全可以整學期都看影片學習、線上交作業， 學生不用進教室（要討論就線上論壇，或者自行約時間），教授不用年年反覆講， 只要期中、期末考實體進行，考試考不好一樣得低分～～就有認證效果。 當然！授課教師要樂於在學生找他的時候提供 office hour 就是了。 我個人是相信這樣可以比傳統教法有更好的教學與學習效果啦，在某些課上。 然後釋放一些教師的人力去開選修、開通識、研發新教材等等，也是好事～～ 學校現在正在研擬讓網路課程（在搭配實體考試的前提下）承認學分的方案， 可想而知這方案的推出會蠻謹慎的。同學們有建議也歡迎多跟教務處說～～ 比如說要是有十萬人連署希望某些網路課程可以承認學分，或許法案或過得比較快 XD === 回文的同學提到翻轉教室，其實跟純線上課程是兩個不同的概念， 不過也無所謂啦 XD。很多同學都很討厭翻轉教室的樣子，我覺得挺可惜的， 其實他就是一種教學方法（或你說理念也行），在有些課有些地方適用， 有些則不適用；而方法無好壞，看用的人有沒有用對而已（跟武器有點像 XD）。 我是覺得他是一種很難上手、很難操作，但練成之後很強大的武器， 只是很多老師還沒有練成而已～～我也不敢說我練成，還有很多改進空間， 但要練到讓大部分同學都同意比傳統教法好，還是辦得到的。 我也知道翻轉教室不會被廣泛採用，因為對教師來說實在沒有比較輕鬆， 而且也不是什麼課都適用。所以同學們不用擔心會全部的課都變成翻轉教室啦～～ 還有很多細節就先不討論了！ -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 219.85.198.230 ※ 文章網址: https://www.ptt.cc/bbs/NTU/M.1483540527.A.9CE.html
