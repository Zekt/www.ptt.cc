手機簡單回覆一下。 分成兩個部分，想念物職治和想用文科轉系。 首先妳說對物治職治很有興趣，你知道這兩個的差別嗎？ 物治有哪四大領域？職治又有什麼領域？ 物治要念六年這件事情有聽說嗎？ （雖說目前是雙軌，但基本上進來老師都會勸你念六年） 文科的部分，正好是本文院企鵝的專長 不知道你文科所謂的好是到怎麼樣的地步？ 還有你想要填的文科究竟是？法？商？還是文學院？ 如果你只是想要轉系的話，強烈建議不要分數到了就填文學院，你沒愛通常也唸不高。不要以為大學的歷史、中文甚至跟外文是同一個等次的，另外還有這幾個系其實常常都有熱愛該領域到一個匪夷所思的地步的人，本魯就遇過好幾個，你想轉系的決心能超過他們嗎？或是真的天資聰穎如斯？（後者也不是沒有） 至於法學院、商學院，裡面頂尖的真的很頂尖（理工宅宅別來亂），也不乏當初二三類跑來念的，你想要拿書卷獎，除了天份也要有點運氣（當然如前所述，天資聰穎至斯另計） 最後則回歸到物職治本身 就我所知，他們的必修通常不太給外系修或旁聽 而你沒修過必修，想轉系也是基本上無望 想轉系其實有個可以考慮的：公衛 聽說大一必修跟該兩系有87成像，而且還在同一棟大樓。 只是我不知道公衛會不會考物理。 但基本上還是希望你指考好好準備，繞路成功的人雖然每年都有，但真的不是一條簡單的路。 以上匆忙回覆。 有誤請用力鞭。 ※ 引述《Yui0626 (喵嗚噠啦)》之銘言： : 第一次在此版發文 : 若違反版規 拜託提醒 : 今年高三 想問大家一些問題 : 剛經歷完學測轟炸 : 本來還算有把握可以考到不錯成績 : 卻被今年數學打趴了QQ : 我是三類生 其實我文科較好 : 但本身對生物跟化學很有興趣 : 也讀得還不錯 就進了理組 : 現在即將面臨指考 : 因為我的物理真的爛得很可怕 : 化學也只有部分單元好 : 看過物理的指考題目 看完頭超暈 : 因此想轉考文組 : 我的問題 : 1.本身是對物治職治很有興趣 : 翻簡章的時候發現職治指考採計國英數甲生 : 如果我除了文組科目 國英數乙史地 : （沒有公民是因為對法比較沒興趣） : 再多準備 數甲 生物 會不會太吃力？ : 還是專心準備文組6科就好了？ : 2.不知道台大的轉系門檻比較不高的傳言是真是假 : 2.不知道台大的轉系門檻比較不高的傳言是真是假 : 有聽到關於GPA的分數要夠 : 請問GPA是如何計算？ : 如果我的發問有任何不妥的地方請告知我！ : 我主要的問題是1. : 畢竟轉系還是要等我真的考上臺大再說哈哈 : 希望有好心人士能幫忙我解惑 : ----- : Sent from JPTT on my Samsung SM-J700F. -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 111.81.8.211 ※ 文章網址: https://www.ptt.cc/bbs/NTU/M.1487737164.A.3D3.html
