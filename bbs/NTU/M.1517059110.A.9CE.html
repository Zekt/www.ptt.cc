※ 引述《brotherho (brotherho)》之銘言： : 標題: Re: [問題] 台大的成績登錄最後期限到底？ : 時間: Sat Jan 27 13:21:09 2018 : : 【成績遲送說明】 : 看到這位同學的情況，其實不斷不斷的在發生，歷屆福利部都與教務處那邊討論過很多次 : ，但也都只能增加一些配套，教務處也對於許多老師晚送成績很頭痛、苦無解方。 : : 學期初，福利部也有針對此事與教務長還有相關教務處同仁開會，目前教務處是有訂定一 : 個期限，期末考週結束後的10天要送成績。 : : 但，教務處做了全面的統計，大約有23％的課程成績還沒送出，換言之在教務處所預先訂 : 下的繳交期限截止後，仍有多達五分之一的課程未繳交同學成績，這個期限形同虛設。 : : 目前針對遲送的，教務處的作法是將遲送老師的名單整理出來，在學校的行政會議上，轉 : 發給各院，希望院系可以提醒老師盡快交成績。 : : 在這個議題上，教務處表示他們目前也 只能柔性勸導，最後的底線還是會要求必須在新 : 學期(2/1)開始前，送出學生成績。 : : 就目前可能權益受影響的個案，教務處也說可能只能向該名老師做個案處理，請他們趕快 : 把個人的成績送出。 : : 成績遲送對學生權益有很大程度的影響，且是申請多數系所雙主修、輔系、轉系時的重要 : 依據，學生將會在每次期末過後陷入何時可以查看、確認成績的不確定，若是對授課教師 : 僅停留在柔性勸導，學生更不可能以修課學生的立場要求老師在超過公告的「柔性死線」 : 後，盡快繳交成績。 : : 福利部在會議中也有提出幾個可能的做法，兩個可能的方案，包括「將成績遲送、課程大 : 綱不完整」列入教師評鑑考評之中 以及 將期末學生教學評鑑移至收到成績後填寫。 : : 就列入教師考評的部分，教務處同仁表示在臺大若要修改此一規定會造成教師極大的反彈 : ，甚至會被 「萬箭穿心 」。 : : 針對期末教學評鑑延後部分，教務處則是認為若是等到成績出來學生才填寫評鑑，恐怕也 : 會因為教師給分的好壞，影響到評鑑的結果，使評鑑不太客觀。 : : 若是有相關改善的建議也歡迎向我們提出，福利部仍會持續討論一個制度性的解決方式， : 盡可能的再來向教務處爭取。 : : 臺大學生會福利部 初步有個構想，拋出來討論看看： 1.將期末評鑑問卷向後遞延至「送交成績截止日」始開放系統。 2.一定比例人數的成績送達後，該課的評鑑問卷才能被填寫。（註） 3-a.學生成績須分別填完該課問卷後，才能分別查看。 3-b.不願填問卷者，5天後開放直接查看，並關閉該課問卷。 4.設定一個「該課已死，請幫教授燒紙」的期限，當天評鑑系統統一關閉，其後送交的成績學生可直接查看，教授得為無法取得評鑑問卷的後果負責。 註：要求「一定比例」原因在於排除只送出少數人成績就讓系統開啟（註中註）；或因為教授仁慈等學生補交作業而承受沒評鑑問卷的不利益。 註中註：學生有特殊狀況可個案處理，教授可先送成績且讓該生提前寫問卷以查看成績。 舉例而言，若1/22是成績截止日、2/13是死亡大限： 若某課成績在1/22以前送達，則1/22當天系統開放時，學生可以直接填寫該課的問卷，立即查看自己的成績。或是1/27該課問卷關閉，直接看成績。 若某課成績在1/25當天送達，則1/25學生才能填寫該課問卷看成績，或是1/30直接查看。 若某課成績在2/14當天送達，則學生可直接查看成績，且該課無問卷可填。 我認為這樣調整可能帶來的好處： 1.課程結束後才進行評鑑，比起在倒數兩周就要你填寫，我覺得前者比較直觀。 2.承上，比起在倒數兩周學生因準備考試忘記填寫，我認為查看成績當下要求填寫，學生填寫意願會較高，而能回收更多評鑑問卷。 3.可以避免教授擔心「評鑑不客觀、甜課才有好分數」的情況發生。（填完才能看，或看了不給填以排除偏頗問卷） 4.教授送交成績時間與課程評鑑問卷開放時間同步，可以納作後續教師評鑑之用。（可以直接看到教授們晚送成績的狀況） 5.承上，就算教評不看教授晚給成績與否，教授在預期學生對於晚到成績的不耐會反映在其評鑑問卷上，而傾向及早送交。 考量教授因課程規劃，將期末報告定在期末考周以後繳交，因批閱需要時間，恐導致留下晚交成績的紀錄，甚至無法取得評鑑問卷，認真的教授可能因而受害… 我認為可以開學前向學校提出交課程大綱並說明（順便約束大綱待補的教授們）。以個案處理的方式，讓教授的成績可以更晚送到且讓學生先填問卷再查看。學生也能在選課時透過課程大綱知悉該課成績可能會晚到，將其納入修課的考量中，避免額外的爭議。 以上是我的想法，還有想到什麼會隨時補充，歡迎討論！XD -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 140.112.251.178 ※ 文章網址:
