BTW, 我說的那兩位同學後來一個跑去醫科一個跑去電機，不知道為啥我發現身邊成績好 的人幾乎都是極端，不是 "超級在意成績一直跟別人比較" 就是 "完全不care我爽就好" 幾乎沒有什麼中庸之人XD : 「我在這門課的學期成績是玖拾壹分，以當年的分數水平來說應該算很高的分數」 : 我想教授最後如願拿到的高分，也是比較出來的。 : 同樣的勵志故事並不少見，高中段考考差了、學測考差了、指考考差了、大學某次段考考差了， : 最後發奮圖強，考上第一名、成功重考、轉系考、轉學考、考上理想校系、比賽得名、不斷拿卷，如願站上了理想的頂端。 : 但是很可悲的，一個人之所以能考好，絕對不是因爲分數到達了某個數值這麼簡單。 : 「祝大家都能考上理想的校系」「祝大家都能考到理想的分數」基本上是廢話，跟「祝大家都能有平均以上」無異。 : 校長對學測和指考生的演說就像是笑笑地告訴大家「現在開始大家自相殘殺，活下來的人才能畢業哦」。 : 大家還能快快樂樂地分享考試訣竅，因爲死亡的階層被隱藏在虛假的標準之中，彷彿只要努力考到那個分數，人人都能進臺大。 : 但學測能考上臺大，並不是因爲我們能夠考到 70 級分好棒棒，而是因爲我們能把另外五萬人的屍體踩在腳底，擠進區區數十人的門檻之中， : 而這個高度需要的分數，剛好跟經驗中相去不遠罷了。 : 反正要到放榜的瞬間，大家才知道自己是被誰殺了， : 只差沒笑笑地對 Elite 們說：「是你們贏了，要連我的份一起活下去哦」 : 我的確相信，優秀與否本來就是比較出來的。 我不覺得。 優秀的人就是優秀，優秀的人是絕對的。 但是這世界資源有限，所以在做抉擇的時候才會需要尋找 "比較優秀" 的人 就像巴伯跟陳三鼎都好喝，妳給我喝我都會說好喝 但是因為巴伯比較好喝，所以我去公館的話會去買巴伯 (理性 勿戰) : 但連臺大對數字的依賴都早以超越實質的知識學習了。 : 人們可以花整整十二年讀一堆自己不知道能拿來幹嘛的垃圾， : 在無謂的填鴨競賽後把數十萬人比下去，最後再回頭對小朋友們自慰： : 「你看，我現在學的這個要用到微積分，所以高中數學要好好學哦」 : 「還好我以前有認真讀書，現在才看得懂」 第一句挺同意的，我覺得不只台大，其實滿多人對於"結果"已經過度重視了 這的確是滿討厭的一點，但我自己覺得也改變不了他們，就警惕自己就夠了 LOL 但是後面那段我完全不能認同XDDD 我想問的是，妳小學就知道未來想要做什麼了嗎？ 因為用妳這套理論，我小學學台灣歷史、學公民、學直笛、打棒球 都 是 垃 圾 那我後悔了嗎？ 幹 當然後悔阿！我她媽浪費了這麼多時間(好啦 打棒球其實挺好玩的) 但有沒有發現這邊出現了矛盾的地方？ 你沒浪費這時間，你又知道妳浪費了時間？ 那些在網路上創作的插畫家，有哪個在拿起畫筆前就知道未來想畫畫？ 那些在網路上寫小說的人，有哪個在接觸文字前就知道自己未來想寫作？ 那些拿著單眼捕捉世界美好的人，有哪個在拿起相機前就知道自己想當攝影師？ 如果是現在的我，絕對是回過頭來跟大一的學弟妹說， "跟妳說 微積分好好學，不要一直翹課，扎扎實實的學，未來很有用der~" 當然妳能說是在自慰，但又何嘗不是避免人們走上錯誤的道路呢？
