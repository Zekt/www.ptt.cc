在工作日的第一天，有很多事我們必須要釐清，作為臺大學生會長，我在此正式呼籲： : 　 : 1. 管中閔教授應出面說明、澄清所有有關疑慮。 : 2. 教育部應儘速作出對臺大校長遴選之裁示。 : 　 : 我們都知道，每件事情不會輕易空穴來風，在查證如此容易的年代，有沒有到廈門、武漢 : 等地兼職，密集班授課、指導研究生、或者只是參與一場講座，不該僅以口號回應，白話 : 文一句就是「有什麼好不能講的」？ : 　 : 我可以理解也許一澄清，就會踢爆了中國高校「一場講座就可以當作兼職」的吹牛皮行徑 : ，但相比起讓臺大不斷落人口實而空懸校長之位，讓臺灣高教龍頭之名持續蒙塵，孰輕孰 : 重？ : 　 : 換言之，如果真的沒有任何不合法之情形，我實在想不到有什麼無法與社會大眾澄清說明 : 的理由。頂多兩種可能：一是有一些講座等比較輕微的活動，曾經未報備過，一解釋就會 : 被持續追打；二是真的有違法兼職情形，可能影響公務員身分而喪失國立大學校長資格。 : 前者解釋、道歉，依照比例原則來作後續處理，便只是如何彌補誠信的問題；後者若日後 : 真的就任臺大校長，也不若總統，可以暫時不為刑事訴究，查一查遲早會發現，這位子恐 : 怕也無法當得太久。 : 　 : 也因此，管中閔教授更應該要儘早出面澄清、解釋當前對於其違法兼職的各種資訊。 : 　 : 臺大有沒有因為新校長未就任所以停擺？那肯定是沒有的，教學、研究、交流及整體的運 : 作不會因此而輕易中止。然而，我們確實也知道人文大樓、新宿舍、高教深耕計畫經費沒 : 人敢批，整個行政團隊「都期待自己明天可以卸任」，我們在推動的學生議題，從英檢、 : 服務學習，到當前校園硬體規劃、主管人事異動被推諉到不知道在哪裡的下一任行政團隊 : ，那倒是屢見不鮮。 : 　 : 臺大正在受傷，主因不會是因為沒有校長，而是因為當可能就任校長的當選人，不斷出現 : 造成社會疑慮的資訊，而我們懸而未決，大家縱然不會批評，也會好奇，「喔，臺大校長 : 是這樣子的人嗎？」 : 　 : 「興訟止謗」在二十年前司法、媒體資源還不普及的時代，也許有威嚇、平息輿論的作用 : ；現在還想用這種招式，恐怕非但不管用，更會招來「是不是不敢講，只敢告，告了之後 : 再說偵查不公開，期待社會大眾忘記」的批評。 : 　 : 現在已經不是批准或否決的問題，現在是應不應該的問題了！ : 　 : 我們不能一方面揮舞大學自治的神聖大旗，一方面又口口聲聲把是非對錯的判斷權交給教 : 育部。 : 　 : 教育部縱然准予核定管中閔教授作為臺大校長，走進校園、走出社會，違法兼職、學術倫 : 理、利益迴避的陰影不會因此散去，又怎能提振過去所謂的臺大學術地位，更遑論念茲在 : 茲的國際聲譽？ : 　 : 此外，兩個多月來，教育部握持行政資源，聲稱開了十多次內部會議，卻一次決定也從未 : 向公眾說明，現在方向明確，大抵不脫遴選程序、學術倫理、兼任外職三個爭議，作出適 : 法性判斷（或表明不能作）不知道還有什麼阻礙，大抵就是不敢擔當，以為在比拼誰「戲 : 棚跤徛久就是你的」。 : 　 : 若有違法、就是違法，若有重大瑕疵、就是重大瑕疵，後續可能產生的司法程序不該是當 :
