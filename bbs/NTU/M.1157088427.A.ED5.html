常見的腳踏車亂停方式: 1.亂插，就像台北車站周邊一樣，有縫就塞 不過由於沒人管理，加上腳踏車本身比較輕，可以搬動 所以很多人就會把兩台車中間小小的空隙搬開，把自己的車塞近兩台車的後輪之間 就自以為完成了停車動作，卻不知道這樣的停車塞住了道路 2.自動延伸停車區域，由於正常的停車格滿了，沒地方停的人就自動停到腳踏車格的旁邊 然後後面的人就會跟著模仿，最後就形成了一條車陣，塞住整條的路 更扯的是有人已經看到塞住路了，還繼續停下去 3.人潮聚集的非停車區域，如小福前的道路每到中午就會被塞住 常見的腳踏車亂停的地區: 1.普通靠近新的地下停車場的地區以及普通大樓週邊 停車場附近有很多人的車亂插，以為這樣就是停進去的 其實這種人只會造成道路變得更崎嶇 2.小福廣場 小福廣場前面是不能停腳踏車的 不過就是有很多人求方便所以就亂停 所以導致那個區域中午的時候連走路都很困難 3.總圖周邊，包括草坪附近和活大 每當期中期末考的時候，就是總圖大爆滿的時候 腳踏車亂插亂塞的到處都是 還可以停到草坪旁邊，自己延伸停車區域，真是天才 還有靠舟山路那端常有人也是自動延伸停車區域，導致路都被塞爆了，行人無法通過 4.計中 因為計中的停車區域小，很多人趕時間所以也是亂插 有時候就整個停滿半個車道 如果停到對面草坪區域的話會被鎖起來，要等他打開才有辦法騎走 5.共同教室 共同教室以前也很糟，不過整修以後有改善 比較礙眼的是違規停在共同教室靠近行政大樓的入口旁邊，賣雜誌的地方 那裡也不是停車區域，不過也有人就是不知道 6.鹿鳴堂 由於這裡有許多人要去買東西，所以對面常常會有人停車 不過我個人認為這裡也算是學校沒有規劃停車格的地方 我覺得校方要付一些責任 7.新體 新體可以停車的地方就是旁邊的那條道路 通常也是亂塞亂插 每當新體有大比賽或活動時，就會從新體自動延伸停車區域 把操場通往新體的路塞住 大概就是這樣 我是b91的，我記得我以前大一大二的時候停車問題沒有那麼嚴重 頂多期末考的時候總圖會大爆滿 但是這幾年就變的很糟糕 我想大家都是台大人，停車的時候應該用一點公德心來想一想 這裡能不能停車，該怎麼停 不要等到被拖到水源校區才開始反省，甚至抱怨學校亂拖車，完全不思考自己的錯 如果看到有人亂停車的時候，可以規勸一下 大多數人就會去找地方停了 ※ 引述《Jman208 (→ 171G8)》之銘言： : ※ 引述《syf0711 (Evelyn)》之銘言： : : 請問 : : 1.校內的腳踏車要停放在哪? : 有幾個地方不要亂停 : 1. 小福樓前面的廣場 請停到旁邊靠近腳踏車店的雨棚內或是舊化學館前面 : 2. 大一女A棟及B棟樓下請不要一次擠太多在那邊 : 3. 計中腳踏車不要超過劃線區域 也不要停在草皮上 : 最常被拖的地方就是這三個 : 基本上不要超出停車格太多 別停草皮 別擋住出口或消防栓 都不太會被拖 : : 2.需要申請停車證嗎? : 去行政大樓拿貼紙就好 不用特地去跑手續申請幹麻 : : 3.若要 那麼還沒申請停車證可以停放在校內嗎? : 可 只是開學之後一段時間會有公告說會拖掉那些沒有貼紙的 請注意 : : 4.另外校內是不是不可以騎機車? : 對 校內汽車都比機車好用 機車不能進校內 : : 謝謝~ -- ╔══════════╗ 我是好人卡 ║ ╔═╗ !!! 比 ║ ║ ●/║好║ <●> 我 ║ 好人專用卡，在別人給你好人卡之 ║<男 ║人║ 女 快 ║ 前，先放出這張卡，可以大幅降低 ║ || ╚═╝ // ║ 壞人卡的攻擊力以及好人的HP損耗 ╚══════════╝ 是今年好人界的最新產品。 koj製作 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 61.228.64.51
