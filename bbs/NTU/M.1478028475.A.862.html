※ 引述《Zekt (Zekt)》之銘言： : 沒有要戰機器學習之神的意思， 我也沒有要戰你的意思 :-) 但是還是想打出來，表達一下個人的看法。 : 但是考試之所以能被稱爲考差而令人難過，從來就不是看分數的，只要出題者想搞學生， : 永遠可以出更難的題目。 心有戚戚焉，太難的題目也沒有參考價值就是這個原因。 : 一個有滿分的考試被「考差了」完全是比較出來的。大家都 20 分時我 59 分，跟大家都 : 89 分時我 59 分就是兩個極端。 : 「我在這門課的學期成績是玖拾壹分，以當年的分數水平來說應該算很高的分數」 : 我想教授最後如願拿到的高分，也是比較出來的。 「分數沒有用，因為是比較出來的」，我相信是奠基在「分數沒有參考價值」的基礎上。 但是重要的考試，命題者都會想辦法避免這個問題，出的題目只要不要全部過於困難或簡單，應該都能有鑑別度，如此一來，分數便擁有了它的參考價值。 故「分數」和「排名」是有著正相關的。 : : 同樣的勵志故事並不少見，高中段考考差了、學測考差了、指考考差了、大學某次段考考 : 差了， : 最後發奮圖強，考上第一名、成功重考、轉系考、轉學考、考上理想校系、比賽得名、不 : 斷拿卷，如願站上了理想的頂端。 : 但是很可悲的，一個人之所以能考好，絕對不是因爲分數到達了某個數值這麼簡單。 我相信考好的同學們，天賦和努力佔了絕大多數啦XD 絕對不是作弊讓自己的分數到達某個數值那麼簡單的事。 我並不是說你作弊，我說的只是那個分數背後代表的是大家的天賦努力運氣最後的總和，我相信每個能考上臺大的同學都不是簡單僥倖的。 : 「祝大家都能考上理想的校系」「祝大家都能考到理想的分數」基本上是廢話，跟「祝大 : 家都能有平均以上」無異。 每個人理想的分數、校系都不一樣啊，又不是每個人都想考進臺大。 不過臺大也就這麼多名額，免不了要競爭啊，社會上不管做什麼都要競爭吧，從搶早上的臺大牛奶到競爭激烈的面試都一樣。 : 校長對學測和指考生的演說就像是笑笑地告訴大家「現在開始大家自相殘殺，活下來的人 : 才能畢業哦」。 : 大家還能快快樂樂地分享考試訣竅，因爲死亡的階層被隱藏在虛假的標準之中，彷彿只要 : 努力考到那個分數，人人都能進臺大。 你的意思是當全部人都考同一個分數的時候，分數就不重要了。 這道理我也懂，但是每年的大考，考出來的人數大致都是常態分佈，也從來沒有一年的均標落在70過啊！ : 但學測能考上臺大，並不是因爲我們能夠考到 70 級分好棒棒，而是因爲我們能把另外五 : 萬人的屍體踩在腳底，擠進區區數十人的門檻之中， : 而這個高度需要的分數，剛好跟經驗中相去不遠罷了。 : 反正要到放榜的瞬間，大家才知道自己是被誰殺了， 考試也是競爭，被殺的人不管是讀書不比別人認真、理解力比別人差一點、或者只是機運比較不好，都是有原因，而不是沒來由的，像是玩天黑請閉眼被投票處決一樣。 不過考不好的人永遠都可以重考啦，敗部復活的概念。 : 只差沒笑笑地對 Elite 們說：「是你們贏了，要連我的份一起活下去哦」 : 我的確相信，優秀與否本來就是比較出來的。 : 但連臺大對數字的依賴都早以超越實質的知識學習了。 但是如果不用分數來評比的話，請問該用什麼來評斷呢？ 既然你相信優秀與否都是比較出來的，那麼分數只不過是量化比較的結果而已。正相關。 : 人們可以花整整十二年讀一堆自己不知道能拿來幹嘛的垃圾， 如果一個人只覺得你是在學用不到的垃圾的時候，他當然只學得到垃圾。 至少我從小到大也在學校學了很多東西啦，不要把課綱編者的苦心都講成垃圾。 你能發深度文，我能發廢文，都要感謝以前學的這些「垃圾」呢。 : 在無謂的填鴨競賽後把數十萬人比下去，最後再回頭對小朋友們自慰： : 「你看，我現在學的這個要用到微積分，所以高中數學要好好學哦」 我認為讀書不僅僅是填鴨學習。 抄襲隔壁的答案比較像是填鴨學習，更何況填鴨填的出來的人也是在某方面上也是贏人了，或許是記憶能力，或許是視力、是演戲的能力XD : 「還好我以前有認真讀書，現在才看得懂」 : 阿，真的這麼想學，十二年前就走進臺大的教室，沒有人會攔你。 你十二年前的志向不會和現在一樣吧。 為什麼讀書就不能被視作志向探索的旅程之一呢？ 因為讀書而找到自己志向的人也大有人在啊。 聽過一個笑話：如果大家小時候的夢想都成真，那月球上就會擠滿了一堆失業的人。 : 看到自己需要補足哪方面的知識，就走進另一間教室，沒有人會攔你。 : 網路更是充滿了無邊的知識，更沒有人會攔你。 網路在教授講的你都聽不懂的時候超好用的XD : 真的知道自己會什麼、不會什麼，分數就是出題者對學生瞭解程度的評估，讓學生在對與 : 錯之間與自我確認。 出太難、出太簡單的考卷，也沒有讓學生自我檢定的能力。 : 先前討論課時跟交換生聊天，聽德國來的交換生一直抱怨臺大的教授（當然，並不是在說 : 田神）如何地把大家都當小孩。 : 雖然我想相信只是剛好她遇到的教授是如此，但連學生們都能以讀書是責任而互相勸勉了 : ，我想她只是來錯大學了吧？ : 還是得勸勸將認真努力讀書考高分作爲責任的人， : 當苦思著要如何努力、如何將眼前的科目都好以負起學生的責任時，代表了要是沒有責任 : ，你根本不想讀書。 讀書真的不是每個人的志向啦。 但是讀書可以讓人獲得基本知識（國民教育的意義）啊，不然大家集體放棄國民教育，文明倒退兩百年也不太好吧。 : 但這麼沉重的責任，實則也無人有義務負擔。 : 到了臺大很多人也終將理解自己花了這麼多時間堆疊起來的不過是作垃圾山，恰好中間有 : 些寶藏。 : 但有些事無法靠認清事實改變。能夠改變自己，卻無法改變結構， : 真正的成功者，永遠都能大聲頌揚著成功。 : Knowledge is free. : 人們不需要踏著失敗者的屍體站上國家的頂點只爲了學到什麼。 : 當然，若是以成爲 Elite 爲目標，就是另一回事了。 踏著失敗者的屍體，想必指的是那些對於自己考了高分沾沾自喜的人吧。 我相信考得好的人都會或多或少有過這種想法——不能否認我自己也有過這種想法——但是要如何從這種心態中調適過來，就是重要的點了。 為什麼很多台大學生都會被認為「自我中心」、「自視甚高」呢？這確實是一個問題。 不過幸好我認識的台大人當中，很少碰到這樣的人。我寧願相信外界對台大的刻板印象真的只是誤解。
