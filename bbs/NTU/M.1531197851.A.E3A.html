嗨你好， 可以抵學分是確定的～～至於一門課能抵通識還是選修，則由相關單位認定： 1. 通識：授課教師、數位學習中心數位課程學分採計委員會、共教中心通識課程委員會，三者都同意則可 2. 選修：授課教師、數位學習中心數位課程學分採計委員會、欲申請採計之學生所屬之系所，三者都同意則可。 換言之針對可抵通識的 coursera 課程，數位學習中心不久後會公告一個清單，全校同學一體適用；針對可抵選修的，則數位學習中心也會公告一個清單，但各系學生的適用性則依其所屬系所而定。 各系所是否同意該系所學生修 NTU coursera 抵選修，不是教務處可以決定或強迫的，教務處只能（也只應該）勸說。同學們若覺得有需求，也可以跟各自的系所反應、讓他們知道喔！ 希望以上有回答到～ ※ 引述《PyTorch (PY火炬)》之銘言： : 請問各位版友 : 這個座談會結束之後 : coursera台大課程確定能抵學分了嗎？ : 小魯現在在上coursera機器學習基石上下 : 人工智慧search, learning : 這樣有沒有機會在107-1的時候折抵3學分呀？ : ＭＬＤＳ好像沒開了 選課好困擾ＱＱ : ※ 引述《rrro (小傑)》之銘言： : : 經過了兩年左右的發想、研擬、推動、溝通、籌備； : : 經過組內會議、教學發展中心與數位學習中心會議、教務處處務會議、 : : 教務會議會前會、教務會議等各個關卡... : : 日前，臺大終於正式通過線上課程的學分採計規定。自 107-1 學期起， : : 「所有」臺大在學學生，在「任何在學期間」自行修習臺大開設（且經授課教師、 : : 數位學習中心、共教中心、各系所等課程委員會認可）的 Coursera 上的 MOOCs 課程， : : 並且通過純線上的考核，「無須付費購買證書或辦理認證」， : : 就可以辦理每人最多三學分的學分採計。 : : 臺大 MOOCs 清單：https://www.coursera.org/taiwan : : 各系所是否認可這些 MOOCs 課程能充抵該系所的選修學分，此刻我們還難以猜測， : : 也無權干涉，但學校這邊如無意外，自 107-1 學期起， : : 應該至少會通過十幾二十門課程可以充抵通識學分。 : : 而且只要大家持續努力，這個清單應該會愈來愈豐富、完整。 : : 申請流程也很簡單。同學只要在通過課程後，隨時去 myNTU 點一點、拉一拉， : : 並且提供自己的 Coursera 帳號，再收封 e-mail 授權檢視該線上課程的成績， : : 數位學習中心的同仁就會（很辛苦地）去 Coursera 後臺檢視同學是否確實完課， : : 並接著接上學校既有的學分認證和成績單登錄流程。 : : === : : 大家或許會有很多問題。為什麼要通過這個規定？為什麼要限 MOOCs？ : : 話說回來，什麼是 MOOCs？為什麼要限三學分？線上成績審核可靠嗎？ : : 數位學習可以取代實體授課，或彌補實體授課的不足嗎？ : : 六月 13 號星期三中午，數位學習中心將在共同 201 舉辦座談會， : : 一方面由我這個組長來說明學校的理念和想法， : : 一方面也請數位學習的先驅黃尹男老師和胡程維同學來分享自己的經驗和建議。 : : 歡迎大家都來吧～～ : : 活動網宣：https://www.facebook.com/ntumooc2017/posts/1734020113358865 ----- Sent from JPTT on my iPhone -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 223.140.13.63 ※ 文章網址: https://www.ptt.cc/bbs/NTU/M.1531197851.A.E3A.html
