標題Re: [食記] 牛髓知味 汀州路紅燒牛肉麵
時間Wed Jun 14 14:30:05 2017
經過 s大的推薦，小弟我今天也去吃了「牛髓知味」 和小弟在台大附近吃過的幾家牛肉麵相比，真的很好吃!!牛肉滷得有嚼勁而不老， 用的麵雖然是普通的細麵，但吸飽湯後，富含湯頭中的香味。對比在後門、前門吃 過的幾間牛肉麵，那種「麵湯分離」的感覺，完全不會在這家店中出現。這家的湯 頭也很有特色，並非普通的滷汁，是略帶辣味的川味。但真的是很淡的辣味XD相信 多數人可以接受。女友是那種連鐵板燒小辣都不能吃的人，但也喝得很開心。老闆 阿嬤說是有加入中藥材一起燉煮的，還很抱歉的和我說因為湯頭熬製不易，不能加 湯。但我覺得這種湯頭比那種標榜「加湯加麵不用錢」的湯頭，濃郁、好喝了不只 一個檔次!! 現在開幕期間的免費小菜一樣有，今天中午有毛豆、豆干、台式泡菜三種。我選了 毛豆，阿嬤並非買外面現成的冷凍毛豆，醃的毛豆用料很札實。吃得出麻油、胡椒 粒、辣椒的香味，非常美味。老闆阿嬤也非常貼心，看到女生去吃麵，會主動問說 「麵少一些，多給妳一塊牛肉好不好? 」(結果女友的多兩塊牛肉，還是被我吃完XDD) 臨走前我和阿嬤說牛肉麵真的非常好吃! 阿嬤很開心地笑了。然後我和阿嬤說台大 有同學很推薦阿嬤的麵，我才來吃的XD阿嬤很不好意思地說「怎麼有這麼好心的人 ，你知道是哪位同學嗎，我要好好謝謝他」但我當然不知道 s大是誰哈哈。 綜合說起來，優點有三 1.湯頭濃郁、牛肉給得很大方 2.小菜入味，和牛肉麵是絕配 3.老闆很熱心可愛 不過店面真的略小、建議可以十一點半或是一點之後去吃，座位比較多。價位略高 (一碗 140)，但應該也算是台北的牛肉麵公道價。尤其是愛吃麵的人，絕對會覺得 很划算! 推薦大家可以光顧看看! ※ 引述《s8705487054 (AAABBB)》之銘言： : 最近經過台大後面汀州路經常會關注一家我看過最小的牛肉麵店「牛髓知味」， : 開在新馬辣與瓦崎之間，大概只有兩坪，小到不可思議，店面不太起眼，很容易錯過。 : 從大約一個多月前剛開店時就有去消費，直到今晚大概也去個五次了。 : 還記得第一天開幕時阿嬤容光煥發，很熱情地推薦自己的牛肉麵好吃， : 也確實滿好吃的，肉很入味大概給個五、六塊，份量也不小， : 開幕時掛的「試賣期間小菜免費」活動，至今依然進行著。 : 還記得一年前有位前輩來公館指名要吃牛肉麵我還真不知帶他去哪吃， : 牛髓知味是我目前在公館吃過最好吃的吧！ : 之後去的幾次，生意都不算太好， : 阿嬤時常面無表情的坐在店內， : 或是帶著疲累沮喪的表情到騎樓下走走、招攬生意， : 每次看到此場景總有些難過， : 今晚她放在外面的一張桌子被正在等待隔壁兩家店的年輕人佔用， : 阿嬤也露出無奈的表情，但卻沒有吭聲，我更心酸了。 : 說真的這家店的確是有些缺點的， : 店面小(店內有4-5個座位，有冷氣，騎樓下有2-3人座位) : 牛肉麵140元價格偏高、只賣紅燒牛肉麵、沒有賣燙青菜或其他食物， : (曾有聽過外帶的顧客詢問阿嬤是否有燙青菜， : 我個人也認為如果有燙青菜會更好) : 儘管牛肉麵味道好，可能也會因為只賣單一產品而失去部分客群。 : 開幕至今客人依然不多，網路上搜尋也幾乎沒有資訊， : 是不是有很多人沒發現這家店呢？ : po出了此文與大家分享，希望讓更多人發現這家店。 : 喜歡紅燒牛肉麵的人可以去試試味道！：） : 店名：牛髓知味 : 地址：台北市中正區汀州路三段297號 (新馬辣隔壁) : 菜單：紅燒牛肉麵$140、自製小菜$30 : (試賣期間送小菜的活動不知道到什麼時候，今晚去還有送~) : 附上牛肉麵與店面照片。 : http://imgur.com/a/TdMM4 : http://imgur.com/a/27Guf -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 140.112.25.99 ※ 文章網址: https://www.ptt.cc/bbs/NTU/M.1497421807.A.1E1.html
